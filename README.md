Execute como Administrador o ATIVAR_ALERTA.bat quando o Windows está bloqueando a execução de scripts por segurança.

Passo 2: Verifique o nome do Outlook no outro PC
O "Novo Outlook" às vezes tem nomes diferentes. No outro computador, abra o Outlook e veja:
O ícone dele na barra de tarefas é o mesmo do seu?
Quando você coloca o mouse em cima do ícone, aparece escrito exatamente "Caixa de Entrada"?
Passo 3: O Antivírus
Se o outro PC tiver um antivírus diferente (como McAfee, Norton ou Kaspersky), ele pode estar matando o processo do PowerShell assim que ele começa. Tente desativar o antivírus por 5 minutos só para ver se o alerta aparece.
